# 批次效应

## 基本信息

- **ID**: `batch-effect`
- **类型**: issues
- **难度**: intermediate
- **标签**: `atac-seq`, `scRNA-seq`, `qc`, `multi-sample`

## 描述

由于实验时间、操作人员、试剂批次等非生物学因素导致的数据系统性差异。

## 详细说明

批次效应是多组学高通量数据中普遍存在的系统性变异，来源于实验操作、时间、试剂批次、测序仪、测序 lane 甚至数据处理流程等非生物学因素。在 ATAC-seq 中，批次效应可能表现为：相同样本在不同批次中产生的 peaks 数量和信号强度差异、PCA 或 t-SNE 聚类图中样本按批次而非生物学分组聚集、差异分析中出现大量假阳性或真阳性信号被掩盖。识别批次效应的方法包括可视化（PCA、UMAP、热图）、统计检验（ANOVA、PERMANOVA）和量化指标（如 silhouette score）。校正策略分为实验设计层面（随机化样本分配、使用内参标准品）和计算层面（ComBat、Harmony、MNN、RUV-seq、PEER 等算法）。需要注意的是，批次校正应在去除技术噪音后、差异分析之前进行，且过度校正可能消除真实的生物学信号。
