# Fragment Size

## 基本信息

- **ID**: `fragment-size`
- **类型**: concepts

## 描述

ATAC-seq 中 DNA 片段的大小分布，反映了核小体占位和开放染色质特征。

## 详细说明

ATAC-seq 的 fragment size 分布通常呈现周期性峰：
- ~50 bp：无核小体区域（NFR），代表开放染色质
- ~200 bp：单核小体区域
- ~400 bp：双核小体区域
通过分析 fragment size 分布可以评估 ATAC-seq 数据质量。
